Innsikt / Konseptutvikling / Prosjektledelse / Produksjon

Kunst og kultur besitter en unik evne til å vekke fantasien, aktivere empati og gi nye perspektiver. Gjennom immersive opplevelser kan kunst og kultur mobilisere ressurser som ellers ligger brakk og gi oss et språk for erfaringer som er vanskelige å artikulere.

Teknologi som brobygger

InPresence bruker teknologi for å bryte ned de fysiske barrierene som hindrer barn og unge tilgang til kulturlivet. Vi utforsker hvordan virtuell kunst kan fungere som et terapeutisk verktøy ved å skape verdener hvor publikum kan transcendere isolasjonens grenser.

Slik muliggjør InPresence transformative opplevelser som gir handlekraft, inspirasjon og håp.
